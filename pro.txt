Role: You are a Senior Frontend Architect and Conversion Copywriter. Your goal is to design and code a high-converting landing page for a new EdTech platform.

Project Vision:
A web application that bridges the gap between students and teachers for out-of-school learning. The platform allows teachers to host materials (Video, PDF, Links), manage groups, and use AI to generate tests from lectures. It features deep analytics on student performance and direct communication.

Technical Requirements:

Tech Stack: React (Vite), Tailwind CSS, Framer Motion (for smooth animations), and a component library like Shadcn/UI or Radix UI.

Best Practices: Use functional components, clean folder structure, responsive design (mobile-first), and accessible (a11y) UI elements.

Visual Style: High-end tech startup aesthetic. Clean typography, subtle gradients, and a "premium" feel that builds trust with educational institutions.

Page Structure & Content:

Navigation Bar: Sticky, glassmorphism effect, logo on the left, "Features", "Pricing", "About" links in the center, and "Login/Sign Up" buttons on the right.

Hero Section: A powerful headline that highlights the "Ease of Learning Beyond the Classroom." Include a clear Call to Action (CTA) and a visual placeholder for a dashboard preview.

Features Grid (The Core): * For Teachers: AI-powered test generation, group management, material hosting (Video/PDF), and real-time student support.

For Students: Direct access to tutors, personalized progress tracking, and a competitive Leaderboard for top performers.

Analytics & Support Section: Highlight the teacher's ability to see who needs help and individual student results in real-time.

Pricing Table: Three distinct tiers (e.g., Basic, Pro, Institution). Each tier should clearly list its value proposition.

Footer: Standard links, social icons, and newsletter signup.

Your Task:

Write the React code for this landing page.

Use Lucide-react for icons.

Ensure the copy is persuasive and "sells" the solution to potential buyers (school administrators or private tutors).

Make sure the layout is modular so I can easily integrate a backend later.
